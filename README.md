📘 Panduan Penggunaan Sistem E-Aspirasi
Aplikasi ini dirancang untuk memudahkan siswa menyampaikan keluhan atau saran terkait sarana sekolah, serta membantu admin/petugas dalam mengelola data tersebut secara praktis.
1. Untuk Siswa (Pelapor)
Akses Halaman Utama: Buka aplikasi, lalu klik tombol Authenticate pada halaman depan.
Registrasi: Jika belum punya akun, klik "Daftar di sini" pada halaman login. Masukkan NIS dan data yang diminta.
Login: Masukan NIS dan Password yang sudah terdaftar, lalu klik Masuk Sekarang.
Kirim Aspirasi: Setelah masuk, gunakan formulir pengaduan untuk menuliskan detail keluhan atau saran mengenai sarana sekolah.

2. Untuk Admin/Petugas (Pengelola)
Admin memiliki kontrol penuh untuk memantau statistik dan mengelola data pengguna.

A. Memantau Statistik (Dashboard)
Buka menu Dashboard untuk melihat ringkasan data.
Di sini kamu bisa melihat jumlah Total Masuk pengaduan dan status respon secara real-time.

B. Mengelola Pengaduan Masuk
Pilih menu Pengaduan Masuk di sidebar.
Kamu akan melihat daftar laporan siswa lengkap dengan statusnya (Waiting, On Process, atau Finished).
Klik pada laporan untuk melihat detail atau memberikan tanggapan/update status.

C. Mengelola Data Siswa (Master Data)
Pilih menu Data Siswa untuk melihat seluruh siswa yang terdaftar di sistem.
Pencarian: Gunakan kolom "Cari NIS atau Nama..." untuk menemukan siswa tertentu dengan cepat.
Menghapus Data: Jika ada data yang tidak valid, klik ikon Tong Sampah (Merah) di kolom Opsi untuk menghapus siswa tersebut dari database.

💡 Tips Penggunaan di HP (Mobile)
Aplikasi ini sudah mendukung tampilan responsif.
Sidebar: Jika menu samping tidak muncul, klik ikon tiga garis (hamburger menu) di pojok kiri atas.
Scroll Tabel: Jika kolom "Aksi" atau "Hapus" tidak terlihat, cukup geser tabel ke arah kanan menggunakan jari kamu.
Keluar Sistem: Selalu klik tombol Logout System di bagian bawah sidebar setelah selesai bertugas untuk menjaga keamanan data.
